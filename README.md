# FISHLIGHT Studio Site

Static v1 concept site for FISHLIGHT AI Studio.

## Open locally

```bash
cd /Users/eugene/Documents/Obsidian仓库/fishlight-studio-site
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173
```

## Deploy on Cloudflare Pages

- Connect this repository to Cloudflare Pages.
- Framework preset: `None`
- Build command: leave empty
- Build output directory: `/`
- Root directory: `/`

- `assets/hero-cinematic-field.png`, `assets/transition-signal-system.png`, and `assets/signal-*.png` are the cinematic site assets.
